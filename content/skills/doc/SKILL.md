# Document

A long-form reading document — report, guide, or spec — as one `index.html`.

## Editorial hierarchy

- Establish one dominant entry point: a title 3–5× the body size.
- Measure: 60–75ch (`max-width: 65ch`). Never justify body text — it creates rivers.
- Three weights only — read, emphasize, announce. Restrained bold (≤2 phrases per
  400 words).
- ALL-CAPS labels always get ≥0.06em letter-spacing.
- Pull-quotes as typographic interrupts, not callout boxes.

Use the active design system's fonts and colors. Real content only — structure
empty sections with composition, not filler.
