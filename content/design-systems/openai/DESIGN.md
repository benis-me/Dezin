# OpenAI

> Category: AI & LLM

## 1. Visual Theme & Atmosphere

Clean, neutral, slightly clinical: white surfaces, near-black text, a single muted
green accent. Restrained and product-like — the UI gets out of the way.

## 2. Color Palette & Roles

- Surfaces: --bg #ffffff, --surface #f7f7f8, --surface-2 #ececf1
- Text: --fg #0d0d0d, --fg-2 #40414f, --muted #6e6e80
- Accent: --accent #10a37f on --accent-fg #ffffff
- Border: --border #e5e5e5, --border-strong #c5c5d2

## 3. Typography Rules

A neutral grotesque sans; mono for code. Body 16px / 1.6; headings restrained. Two weights.

## 4. Component Stylings

Buttons: green primary or --surface ghost, radius 8px. Chat bubbles: --surface fill,
flat. Inputs: 1px border, focus → --accent. Code blocks: dark tint with mono.

## 5. Layout Principles

4px grid; centered column max 768px; calm spacing. Radius 6 / 8 / 12px.

## 6. Depth & Elevation

Flat; light shadow on menus only.

## 7. Do's and Don'ts

Do: neutral surfaces, one muted-green accent, calm rhythm. Don't: gradients, neon, multiple accents, busy chrome.

## 8. Responsive Behavior

Breakpoints 640 / 1024px; single column under 640; targets ≥24px.

## 9. Agent Prompt Guide

bg #fff / fg #0d0d0d / accent #10a37f / border #e5e5e5. Example: "A clean assistant UI: flat bubbles, green send button, neutral grays, mono code."
